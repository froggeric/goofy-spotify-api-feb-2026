## Features

- tracks listening history
- collects recommendations and new releases
- has a large set of filters
- imports <a target="_blank" href="https://chimildic.github.io/goofy/#/en/addon?id=FM-radio-track-import">FM radio</a> and Last.fm
- supports advanced search
- allows dynamic queue changes
- works on schedule and with <a target="_blank" href="https://github.com/Chimildic/goofy/discussions/124">Tasker</a> events
